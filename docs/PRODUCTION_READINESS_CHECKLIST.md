# Production Readiness Checklist

- [ ] All critical bugs fixed.
- [ ] All high findings fixed or accepted with risk sign-off.
- [ ] Workflow linting enabled in CI.
- [ ] Integration tests run for happy/edge/failure paths.
- [ ] On-call runbook approved.
- [ ] Alerting configured for failed schedules.
- [ ] Concurrency and idempotency controls enabled.
- [ ] Security review complete for workflow changes.
- [ ] Go/No-Go review signed by engineering + operations.
