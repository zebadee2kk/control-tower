# Best Practices Comparison

## Current vs Best Practice

1. **State management**
   - Current: label convention only
   - Best: explicit transition policy + reconciliation

2. **Automation reliability**
   - Current: ad hoc scripts, limited error handling
   - Best: shared helpers, retries, typed payload checks

3. **Security posture**
   - Current: basic token scope only
   - Best: protected workflow files, actor trust validation, abuse controls

4. **Operational excellence**
   - Current: no SLOs/alerts
   - Best: dashboard, alert routing, incident runbook, postmortems
