# Gold Vault

Distilled, high-value outputs from conversations and research.

## Purpose

**Raw conversations stay local.** Only polished, reusable insights come here.

## What Goes Here

✅ Research memos  
✅ Architecture decisions  
✅ Lesson learned summaries  
✅ Reusable templates  
✅ Best practices discovered  

## What Stays Out

❌ Raw chat exports  
❌ Rough brainstorming  
❌ Sensitive data  
❌ Incomplete analysis  
❌ Tool-specific outputs without context  

## File Naming Convention

```
YYYY-MM-DD-topic-name.md
```

Example: `2026-02-24-github-workflow-patterns.md`
